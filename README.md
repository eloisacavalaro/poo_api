# Enum - Unidades Federativas

Aplicação Java desenvolvida para praticar o uso de `enum`, associando estados brasileiros às suas respectivas regiões.

## Funcionalidades

- Listagem das UFs disponíveis
- Busca de UF por sigla
- Consulta de estados por região
- Tratamento de entrada inválida

## Objetivo

Praticar:

- Criação e uso de Enum
- Métodos dentro de Enum
- Listas e filtros
- Tratamento de exceções
